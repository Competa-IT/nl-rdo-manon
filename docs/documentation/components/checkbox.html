Selectievak - checkbox

Benodigde stappen:

  1. Voeg de benodigde bestanden toe aan het project. Voor een overzicht van de benodigde en optionele bestanden zie: Bijbehorende bestanden.
  2. Plaats de <input> toe van het type type="checkbox". Voorbeeld: input type="checkbox">
  3. Voeg direct daaropvolgend het bijbehorden <label> toe.

Tip: Om checkboxes een eigen gekozen kleur mee te geven kan gebruik gemaakt worden van accentkleur op formulierelementen.

Voorbeelden

Importeer component via NPM

CSS-voorbeeld:

              
@use "@minvws/form-checkbox";

Instelbare variabelen

  • margin-right
  • vertical-align
  • width
  • height

Instelbare variabelen

Let op: De uitlijning van horizontaal uitgelijnde tags werken alleen in combinatie met de class horizotal-view.

--form-checkbox-label-width width auto -
